# Claude Code CLI 教學 — 總覽

## 一句話說明

> **Claude Code CLI** 是 Anthropic 官方的終端機 AI 助手：在終端機呼叫 `claude`，讓 AI 直接讀你的程式碼、幫你改程式、執行指令，不需要開瀏覽器。

---

## 為什麼要用 Claude Code CLI？

| 情境 | 用瀏覽器版 Claude | 用 Claude Code CLI |
|------|-------------------|--------------------|
| 修改本機程式碼 | 要自己複製貼上 | AI 直接讀寫你的檔案 |
| 執行終端機指令 | 辦不到 | AI 可以幫你跑指令 |
| CI/CD 自動化 | 不方便 | 非互動模式一行搞定 |
| 已經在終端機做事 | 要切換視窗 | 同一個地方就能問 AI |
| 長對話要繼續 | 重新打很麻煩 | `--continue` 接著上次講 |

**簡單講**：CLI 讓 AI 有辦法真的「動你的專案」，不只是給你建議。

## 前置需求

- 已安裝 **Node.js 18+**（`node --version` 確認）

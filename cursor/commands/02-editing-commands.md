# 編輯相關指令

## Cmd+K 內聯編輯

**操作**：選中一段程式碼 → 按 `Cmd+K`

| 特點 | 說明 |
|------|------|
| 不開 Chat | 在編輯器內直接修改，不佔用側邊面板 |
| 局部修改 | 針對選中區塊做變更，範圍明確 |
| 終端也可用 | 在終端中按 `Cmd+K` 可生成 shell 指令 |

**適合**：改變數名、加註解、小範圍重構、修正語法。

---

## Tab 智能補全

| 操作 | 說明 |
|------|------|
| `Tab` | 接受 AI 的補全建議 |
| `Esc` | 拒絕補全 |

**特點**：
- 自動觸發，無需額外按鍵
- 支援多行、函數級別的整段補全
- **Auto-import**：自動偵測並加入缺少的 import
- 會學習你的編輯習慣，預測下一步

---

## Cmd+I 內聯 Agent 編輯

**操作**：在編輯器中按 `Cmd+I`

- 直接啟動 Agent，針對目前檔案或選中內容進行編輯
- 等同於在 Chat 中切到 Agent 模式，但操作更聚焦

---

## 編輯指令對照

| 指令 | 觸發方式 | 修改範圍 |
|------|----------|----------|
| `Cmd+K` | 選中 + 快捷鍵 | 僅選中區塊 |
| `Cmd+I` | 快捷鍵 | 可跨檔案、執行終端 |
| `Tab` | 自動出現 | 單行或小段補全 |

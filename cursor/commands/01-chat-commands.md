# Chat 對話指令

## 開啟與聚焦 Chat

| 指令 | 說明 |
|------|------|
| `Cmd+L` | 開啟 Chat 面板，或將焦點移到 Chat 輸入框 |
| `Cmd+Alt+L` | 查看歷史對話記錄 |

---

## 模式切換：Ask vs Agent

Chat 有兩種模式，在輸入框上方可切換：

| 模式 | 行為 | 適用場景 |
|------|------|----------|
| **Ask** | 僅回答問題，不執行操作 | 解釋程式碼、詢問概念、給建議 |
| **Agent** | 可搜尋檔案、編輯程式碼、執行終端 | 重構、修 Bug、加功能 |

**發送方式**：`Cmd+Enter` 在 Agent 模式下發送並執行。

---

## 將選中內容加入 Chat

| 指令 | 說明 |
|------|------|
| `Cmd+Shift+L` | 將目前選中的程式碼或文字加入 Chat 作為上下文 |

**使用時機**：想針對某段程式碼提問時，先選中 → 按快捷鍵 → 輸入問題。

---

## 圖片輸入

- 直接**貼上截圖**到 Chat 輸入框
- 支援多張圖片、多模態分析
- 適合：UI 還原、錯誤畫面、設計稿對照

---

## 新對話與工具開關

| 指令 | 說明 |
|------|------|
| `Cmd+N` | 開啟新對話（建議新話題時使用，避免上下文膨脹） |
| `Cmd+.` | 切換 Agent 工具開關（控制 Agent 能否執行操作） |

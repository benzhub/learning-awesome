# Cursor 快捷鍵速查表

## Chat 與對話

| 快捷鍵 | 功能 |
|--------|------|
| `Cmd+L` | 開啟 / 聚焦 Chat |
| `Cmd+Alt+L` | 查看歷史對話 |
| `Cmd+Shift+L` | 將選中內容加入 Chat |
| `Cmd+N` | 開啟新對話 |
| `Cmd+Enter` | Agent 模式發送 |
| `Cmd+.` | 切換 Agent 工具開關 |

---

## 編輯

| 快捷鍵 | 功能 |
|--------|------|
| `Cmd+K` | 內聯快速編輯（選中後） |
| `Cmd+I` | 內聯 Agent 編輯 |
| `Tab` | 接受 Tab 補全 |
| `Esc` | 拒絕 Tab 補全 |

---

## 平台差異

| 平台 | Cmd | Ctrl |
|------|-----|------|
| macOS | `Cmd` | — |
| Windows / Linux | — | `Ctrl` |

macOS 的 `Cmd` 在 Windows/Linux 對應為 `Ctrl`。

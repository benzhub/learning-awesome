# Cursor CLI 教學 — 總覽

## 一句話說明

> **Cursor CLI** 是終端機裡的 AI 助手：不用開 Cursor 視窗，在終端機就能請 AI 幫你寫程式、改程式、回答問題。

---

## 為什麼要用 CLI？

| 情境 | 用 Cursor 視窗 | 用 Cursor CLI |
|------|----------------|----------------|
| 在伺服器上改 code | 要 SSH + 開遠端 | 終端機直接下指令 |
| 寫腳本、CI 自動化 | 不方便 | 非互動模式一鍵跑 |
| 已經在終端機做事 | 要切到 IDE | 同一視窗就能問 AI |
| 喜歡鍵盤流 | 滑鼠點來點去 | 全部指令搞定 |

**簡單講**：CLI 讓 AI 跟著你的工作環境走，而不是你一定要進 IDE。

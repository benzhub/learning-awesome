# 01 — 什麼是 Elementor？

## 一句話

> Elementor 是一個 WordPress **拖放式頁面編輯器**：不用寫 HTML/CSS，直接在畫面上拖積木、排版、看預覽。

---

## 沒有 Elementor 之前

建一個 WordPress 頁面，你需要：

1. 在後台寫文字
2. 靠主題（Theme）決定排版
3. 改樣式？要去改 CSS 或 PHP

換句話說，**所見不一定是所得**。

---

## 有了 Elementor 之後

你打開「用 Elementor 編輯」，畫面直接變成這樣：

```
┌─────────────────────────────────────────────────────┐
│  左側：Widget 面板              右側：即時預覽       │
│  ┌─────────────┐               ┌──────────────────┐  │
│  │ 標題         │               │  我的網站標題     │  │
│  │ 文字         │               │  ──────────────  │  │
│  │ 圖片         │   ← 拖放 →   │  這裡是文字內容   │  │
│  │ 按鈕         │               │                  │  │
│  │ 影片         │               │  [立即購買]       │  │
│  │ ...          │               │                  │  │
│  └─────────────┘               └──────────────────┘  │
└─────────────────────────────────────────────────────┘
```

左邊選 Widget、拖到右邊，**馬上看到結果**。

---

## Elementor 的核心概念

| 名詞 | 意思 | 比喻 |
|------|------|------|
| **Widget** | 一個可拖放的功能積木 | 樂高積木 |
| **Section / Container** | 放 Widget 的容器（一排） | 積木底板 |
| **Control** | Widget 的設定項目（文字、顏色…） | 積木的顏色選項 |
| **Panel** | 左側的設定面板 | 遙控器 |

---

## Elementor 分免費版和 Pro 版

| 功能 | 免費版 | Pro 版 |
|------|--------|--------|
| 基本 Widget（文字、圖片、按鈕…） | ✅ | ✅ |
| 進階 Widget（表單、彈窗、Slider…） | ❌ | ✅ |
| Theme Builder（頁首、頁尾…） | ❌ | ✅ |
| **自訂 Widget（這套教學的主題）** | ✅ | ✅ |

> 好消息：**自己寫 Widget 不需要 Pro 版**，免費版就夠了。

---

## 為什麼要「自己寫 Widget」？

內建的 Widget 再多，總有不夠用的時候：

- 你有一個特殊的計算機功能，要整合進頁面
- 你的客戶想在編輯器裡直接設定某個客製區塊
- 你想把重複的 HTML 結構包成一個可重用的積木

這時候，**自訂 Widget** 就是答案。

---

## 小結

- Elementor = 拖放式編輯器，左邊積木、右邊預覽
- Widget = 編輯器裡的每一個積木
- 自己寫 Widget 不需要 Pro 版
